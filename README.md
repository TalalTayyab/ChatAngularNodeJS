# ChatAngularNodeJS ** [6]
A sample chat application using AngularJS, NodeJS and Socket IO.

 The following libraries were used on server side

    Node JS
    Vash
    Socket IO
    Underscore
    
   The following libraries were used on client side

    Angular JS
    Bootstrap
    Socket IO client

Node JS unit testing using

    Mocha
    Chai
    Rewire
